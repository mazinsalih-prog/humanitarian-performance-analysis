# Humanitarian Program Performance Analysis

## 📊 Project Overview
This project analyzes humanitarian program performance across sectors, states, and donors using Python and Power BI.

The objective is to transform raw program data into actionable insights that support Monitoring, Evaluation, and Learning (MEL) processes and evidence-based decision-making.

---

## 🎯 Objectives
- Assess performance across humanitarian sectors
- Compare performance across states and donors
- Identify high-performing and underperforming programs
- Support data-driven program improvement

---

## 🛠️ Tools & Technologies
- Python (Pandas)
- Google Colab
- Power BI
- Matplotlib

---

## 📁 Dataset Description
The dataset simulates humanitarian program data and includes:
- Project information
- Indicators (targets vs actuals)
- Activities
- Beneficiaries
- Geographic and sector classifications

---

## 📊 Key Insights

### Sector Performance
- Health sector shows the strongest performance
- FSL sector shows the weakest performance and requires attention

### Geographic Performance
- North Darfur is the best-performing state
- West Kordofan is the lowest-performing state

### Overall Performance
- 50% of indicators are On Track  
- 30% require attention  
- 20% are critical
